PROJECT:

Destiny MetaTrends

DETAILS:

Destiny MetaTrends is a highly developed data parsing tool that defines weapon and player trends in a way unavailable elsewhere. It does this by compiling an ever increasing library of player vs. player post-game reports which offer highly detailed data of every event occuring in a particular game.

PROJECT:

Flexbox Sandbox

DETAILS:

Focusing on one of the most fundamental and simple layout modules provided natively in CSS, Flexbox Sandbox offers randomly generated puzzles for users to solve. Every puzzle differs in the number of parent and child elements generated and which Flexbox properties are adjusted for each element to provide unlimited practice for anyone desiring to learn this method of website layout.

PROJECT:

Global Location Search

DETAILS:

Global Location Search accesses the databases of Google, Wikipedia, and NewsAPI to retrieve a well-rounded breadth of information for any location searched.
